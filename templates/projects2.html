Recommendation system

This project consists of 3 types of recommendation system.

  • Most popular product: Based on the ratings received from the customers I created a model that generates the top 10 selling products recommended to every user.
  • Collaborative filtering: Consider the user journey of various users. With this data, I created a correlation matrix that enables user to find the product resembling the products bought from other users.
  • Model to find products for a new business: In this project we convert product description with TFIDVectorizer and create cluster and recommend products to customers based on the clusters resembling their keyword searched.
